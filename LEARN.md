# TradeSense Mastery: Full-Stack Skill Roadmap

Ei list-ti apnar **TradeSense** project-er architecture (README.md & ROADMAP.md) ebong apnar future mobile app plan-er opor vitti kore banano hoyeche. Ei 21-ti skill sikhle apni ekjon professional Full-Stack Developer hoye uthben.

---

## 🏗️ Phase 1: The Foundation (Web Basics)
- [ ] **1. HTML5:** Web-er structure toiri korar jonno.
- [ ] **2. CSS3:** Basic design ebong layout-er jonno.
- [ ] **3. JavaScript (ES6+):** Main logic, API calling, ebong functionality-r jonno.
- [ ] **4. TypeScript (TS):** JavaScript-ke aro secure ebong error-free korar jonno (Apnar project-er main language).
- [ ] **5. Tailwind CSS:** Khub druto ebong modern design toiri korar CSS framework.

## 🚀 Phase 2: Frontend & Frameworks (User Interface)
- [ ] **6. React.js:** Component-based UI toiri korar library.
- [ ] **7. Next.js 14 (App Router):** Apnar project-er main framework (Server Components, Routing, Server Actions).
- [ ] **8. Node.js:** Server-side environment jekhane JavaScript run hoy.
- [ ] **9. REST API:** Frontend ebong Backend-er moddhe data r anagona korar system.

## 🗄️ Phase 3: Database & Data Management (Backend Layer)
- [ ] **10. SQL:** Database-er sathe kotha bolar language.
- [ ] **11. PostgreSQL:** TradeSense-er main relational database system.
- [ ] **12. Drizzle ORM:** TypeScript diye shohoje database control korar tool.
- [ ] **13. NextAuth.js (Auth.js v5):** Login, Google Login, ebong security system.
- [ ] **14. Zustand:** Website-er ek page theke onno page-e data mone rakhar tool.
- [ ] **15. TanStack Query (React Query):** API theke data fetch ebong cache korar modern tool.

## 🛠️ Phase 4: Tools & Deployment (Production Ready)
- [ ] **16. Git:** Code-er history ebong version control korar jonno.
- [ ] **17. GitHub:** Code online-e save rakha ebong onno developer-er sathe kaj korar jonno.
- [ ] **18. Vercel:** Website live ba host korar platform.
- [ ] **19. Supabase:** Database host korar platform.

## 📱 Phase 5: Future Expansion (Mobile Application)
- [ ] **20. React Native:** JavaScript/React diye mobile app bananor library.
- [ ] **21. Expo:** React Native-ke shohoje run ebong deploy korar framework.

---

## 💡 How to Learn Fast?
1. **W3Schools:** Basic syntax (HTML, JS, SQL, TS) er jonno use korun.
2. **YouTube:** Prottekta topic-er nam likhe "Masterclass" ba "Crash Course" search korun (e.g., *Next.js 14 Masterclass*).
3. **AI (Gemini/ChatGPT):** Prottekta video dekhar por bujhte somossya hole AI-ke prosno korun.
4. **Practice:** Prottekta skill shekhar somoy choto choto mini-project banan.

**TradeSense Phase 4 (Daily Insight MVP) shuru korar jonno 1 theke 15 porjonto skill gulo thaka dorkar.**
